## Purpose

This directory serves to:

1. **Comply with license requirements** - Most open source licenses require including the license text
2. **Provide reference** - Easy access to license terms for developers and users
3. **Demonstrate compliance** - Show that the project respects intellectual property rights

## Distribution

When distributing this project (source code or binary):

1. **Include this directory** - Copy the entire `licenses/` directory
2. **Reference in NOTICE** - Point to this directory in the NOTICE file
3. **Include LICENSE** - Include the main LICENSE file in the root
4. **Include NOTICE** - Include the NOTICE file in the root

---

## Verification

To verify license compliance:

1. Check that all license files are present
2. Verify that each dependency's license is represented
3. Ensure license texts are complete and unmodified
4. Cross-reference with dependencies in `build.gradle.kts`

---

## Updating

When adding new dependencies:

1. Identify the license type
2. Add the license text to this directory (if not already present)
3. Update this README with the new license
4. Update OPEN_SOURCE_USAGE.md with dependency details
