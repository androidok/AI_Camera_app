# CameraX-AI 智能摄影助手 - 安装说明

## 1. 系统要求

| 类别 | 最低要求 | 推荐配置 |
|------|----------|----------|
| **操作系统** | Android 7.0 (API 24) | Android 10.0 (API 29) 或更高 |
| **CPU** | 4 核心 1.5GHz | 8 核心 2.0GHz 或更高 |
| **内存** | 2 GB RAM | 4 GB RAM 或更高 |
| **存储空间** | 100 MB 可用空间 | 500 MB 可用空间 |
| **相机** | 后置摄像头（至少 8MP） | 双摄像头或更高 |
| **网络** | 无强制要求（AI 模型本地运行） | 可选（用于在线更新） |

## 2. 安装前准备

### 2.1 设备设置

1. **启用未知来源安装**
   - 打开设备 **设置** → **安全** → **未知来源**
   - 允许安装来自 "此来源" 的应用
   - 对于 Android 8.0+：
     - 打开 **设置** → **应用和通知** → **特殊应用访问** → **安装未知应用**
     - 选择您的文件管理器应用，开启权限

2. **检查存储空间**
   - 确保设备有至少 100 MB 可用存储空间
   - 建议清理临时文件以释放空间

3. **确保电池电量**
   - 安装过程需要稳定的电力供应
   - 建议电池电量至少 20% 以上

## 3. 安装步骤

### 3.1 方法一：直接安装（推荐）

1. **获取 APK 安装包**
   - 从官方发布渠道下载最新版本的 `AI_Smart_Camera_v1.0.0.apk`
   - 验证 APK 文件完整性（可选）

2. **执行安装**
   - 打开文件管理器，找到下载的 APK 文件
   - 点击 APK 文件，系统会提示"安装应用"
   - 点击 **安装** 按钮
   - 等待安装过程完成（通常 10-30 秒）

3. **完成安装**
   - 安装完成后，点击 **打开** 启动应用
   - 或在应用列表中找到 "AI Smart Camera" 图标

### 3.2 方法二：使用 ADB 安装（开发人员）

1. **准备环境**
   - 安装 Android SDK Platform Tools
   - 启用设备 USB 调试模式

2. **执行命令**
   ```bash
   adb install -r AI_Smart_Camera_v1.0.0.apk
   ```

3. **验证安装**
   ```bash
   adb shell pm list packages | grep com.aicamera.app
   ```

## 4. 首次启动配置

### 4.1 权限请求

首次启动时，应用会请求以下权限：

| 权限 | 用途 | 必要性 |
|------|------|--------|
| **相机** | 拍摄照片、实时预览 | 必需 |
| **存储** | 保存照片、读取相册 | 必需 |
| **位置** | 地理标记（可选） | 可选 |

### 4.2 模型初始化

1. **首次启动时**，应用会自动：
   - 初始化 ML Kit 场景识别模型
   - 加载 MobileNetV2 图像优化模型
   - 检查模型完整性

2. **初始化时间**：约 10-15 秒（首次启动）

## 5. 验证安装

### 5.1 功能验证

1. **相机功能**
   - 打开应用，验证相机预览正常
   - 检查构图辅助线显示
   - 测试场景识别功能

2. **AI 功能**
   - 进入编辑模式，测试 AI 增强功能
   - 进入裁剪模式，测试 AI 智能裁剪

3. **存储功能**
   - 拍摄一张照片，验证保存到相册
   - 从相册选择照片，验证编辑功能

### 5.2 版本验证

1. **检查版本号**
   - 打开应用 → **设置** → **关于**
   - 确认版本号：v1.0.0

2. **检查模型版本**
   - 打开应用 → **设置** → **AI 模型**
   - 确认 ML Kit 版本：17.0.7
   - 确认 MobileNetV2 模型版本：v1.0

## 6. 常见问题与解决方案

### 6.1 安装失败

| 错误信息 | 可能原因 | 解决方案 |
|----------|----------|----------|
| 解析软件包时出现问题 | APK 文件损坏 | 重新下载 APK 文件 |
| 存储空间不足 | 设备空间不够 | 清理存储空间 |
| 应用未安装 | 权限不足 | 启用未知来源安装 |
| 与现有应用冲突 | 版本冲突 | 先卸载旧版本 |

### 6.2 运行问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 相机黑屏 | 权限未授予 | 重新授予相机权限 |
| AI 功能无响应 | 模型未加载 | 重启应用 |
| 应用崩溃 | 内存不足 | 清理后台应用 |
| 识别速度慢 | 设备性能不足 | 关闭其他应用 |

## 7. 卸载说明

### 7.1 标准卸载

1. **方法一**：长按应用图标 → 拖动到 "卸载" 区域
2. **方法二**：设置 → 应用 → AI Smart Camera → 卸载

### 7.2 完全清理

卸载后，建议清理以下数据：

1. **缓存数据**：设置 → 存储 → 缓存数据 → 清除
2. **模型文件**：默认存储在 `Android/data/com.aicamera.app/models/`
3. **照片数据**：保留在相册中，需手动删除

## 8. 版本更新

### 8.1 自动更新

1. **检查更新**：应用 → 设置 → 关于 → 检查更新
2. **下载安装**：按照提示下载并安装新版本

### 8.2 手动更新

1. 下载新版本 APK
2. 直接安装（会覆盖旧版本）
3. 保留应用数据和设置
